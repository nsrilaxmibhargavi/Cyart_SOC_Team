# Phishing Investigation Checklist

When a suspected phishing email is reported, analysts should follow this checklist:

- [ ] Verify sender email address
- [ ] Check email header information
- [ ] Inspect suspicious links in the email
- [ ] Check link reputation using VirusTotal
- [ ] Analyze attachments for malware
- [ ] Identify users who received the email
- [ ] Determine if any user clicked the link
- [ ] Block malicious domains or IP addresses if confirmed
- [ ] Notify affected users and security team


# CyArt SOC Internship – Week 2

This repository contains the practical work completed during Week 2 of the CyArt SOC Internship program.

The tasks focus on Security Operations Center (SOC) workflows including alert management, incident response documentation, alert triage, digital evidence preservation, and a full attack-to-response simulation.

## Repository Structure

### 1. Alert Management
Contains files related to alert classification and prioritization using CVSS scoring and SOC alert workflows.

Files include:
- Alert classification spreadsheet
- Escalation email example
- TheHive incident ticket
- Dashboard screenshot

### 2. Response Documentation
Includes documentation used during incident investigations.

Files include:
- Incident response template
- Investigation log
- Phishing investigation checklist
- Post-incident analysis summary

### 3. Alert Triage
Demonstrates the process of analyzing security alerts and validating threat intelligence indicators.

Files include:
- Alert triage simulation dataset
- OTX threat intelligence validation summary

### 4. Evidence Preservation
Shows how digital forensic evidence is collected and documented during security investigations.

Files include:
- Evidence documentation record
- Hash verification report
- Sample network connection data
- Simulated memory dump file

### 5. Capstone Project
A complete simulated SOC workflow including attack detection, investigation, response, and reporting.

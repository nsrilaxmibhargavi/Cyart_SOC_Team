# CyArt SOC Internship – Week 3

This repository contains the practical work completed during **Week 3 of the CyArt SOC Internship program**.

The tasks focus on **advanced SOC analysis techniques**, including log correlation, threat intelligence integration, incident escalation workflows, alert triage using intelligence sources, digital evidence preservation, and a full Security Operations Center (SOC) investigation simulation.

---

# Repository Structure

## 1. Advanced Log Analysis

This section focuses on analyzing and correlating security logs to identify suspicious behavior and potential security incidents.

Files include:

- Log correlation dataset  
- Anomaly detection analysis notes  
- GeoIP log enrichment summary  

These files demonstrate how SOC analysts identify abnormal activity patterns using log analysis techniques.

---

## 2. Threat Intelligence Integration

This section demonstrates how threat intelligence platforms are used to enrich alerts and validate suspicious indicators.

Files include:

- Threat intelligence feed integration summary  
- Alert enrichment dataset  
- Threat hunting analysis report  

Threat intelligence sources such as **AlienVault OTX and VirusTotal** were referenced to evaluate suspicious indicators.

---

## 3. Incident Escalation

This section documents the escalation workflow used when security incidents require deeper investigation by higher-tier SOC analysts.

Files include:

- TheHive escalation case documentation  
- Situation report (SITREP)  

These documents simulate the process of escalating incidents from **Tier 1 SOC analysts to Tier 2 analysts** for further investigation.

---

## 4. Alert Triage with Threat Intelligence

This section demonstrates the process of analyzing security alerts and validating indicators of compromise (IOCs) using external intelligence sources.

Files include:

- Alert triage simulation dataset  
- IOC validation summary  

This exercise shows how SOC analysts prioritize alerts and determine whether they represent real threats or false positives.

---

## 5. Evidence Preservation

This section demonstrates how digital evidence is collected and documented during security investigations.

Files include:

- Evidence documentation records  
- Hash validation report  
- Sample network connection data  

Proper evidence handling ensures that collected data maintains integrity for forensic analysis.

---

## 6. Capstone Project

This section contains a complete **SOC workflow simulation** demonstrating the lifecycle of a cybersecurity investigation.

The simulation includes:

- Attack detection  
- Security log investigation  
- Threat intelligence validation  
- Incident containment  
- Escalation workflow  
- Final incident reporting  

The capstone project demonstrates how SOC analysts detect and respond to potential security incidents in real-world environments.
